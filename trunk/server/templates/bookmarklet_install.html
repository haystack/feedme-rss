To install the FeedMe bookmarklet, drag the following link onto your bookmark bar:
Share with FeedMe

Do you use NoScript? If so, you can get the FeedMe bookmarklet working by following these instructions:

  1. Right click on the NoScript icon in the bottom right of your browser. Choose "Options..." and then go to the XSS tab. In the list of Anti-XSS Protection Exceptions, add the regex:
    ^http://feedme.csail.mit.edu:8000/[^"<>\?%]+$
  2. Still in the options pane, go to the Whitelist tab. In the textbox, add the following web sites:
    http://feedme.csail.mit.edu:8000
    googlecode.com
